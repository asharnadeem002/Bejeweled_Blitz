Instructions for the game:


1-After the game page appears, press enter to start the game.

2-Use up/down/left/right cursor keys to move the elements selecting box.

3-Press enter on the element you want to swap.

4-Now press "A"(for swap with left element), "S"(for swap with lower element), "D"(for swap with right element) and 
"W"(for swap with upper element).

5-After you press enter, the game shifts to swapping mode. If you want to go back to selecting any other element,
press "F".

6-For each successful swap of 3 elements, you get 10 points.

7-Time limit for the game is 1 minute after which the game ends. If you want to close the game, press "F".
